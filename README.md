# FlashLiquidity Portals

FlashLiquidity cross-chain interoperability and governance executors contracts

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Full Test Suite

`yarn test`

## Run Unit Tests

`yarn unit-test`

## Run Fuzz Tests

`yarn fuzz-test`

## Run Invariant Tests

`yarn invariant-test`

## Run Coverage

`yarn coverage`

## Run Slither

`yarn slither`
